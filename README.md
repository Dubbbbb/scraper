## Git clone
* Run
    ```
    git clone https://github.com/Dubbbbb/scraper.git
    ```

## Installation Instructions

**Package Installer**

* Install BeautifulSoup
* ```
    pip install bs4
    ```
* Install requests
* ```
    pip install requests
    ```
  
# Run applications - Scraper
* Run
    ```
    python scraper.py
    ```
